# Wasteland Contributing Guidelines

Thank you for your interest in contributing to Wasteland! We welcome contributions from the community to help improve the project. Below are some guidelines to help you get started.

## How to Contribute

1. Fork the repository and create a new branch for your feature or bug fix.
2. Install [Vite+](https://viteplus.dev/)
3. Make your changes and open a pull request with a clear description of your changes and the problem they solve.

## Code of Conduct

1. Be respectful and considerate to other contributors.
2. Follow the project's coding style and guidelines.
3. Ensure your contributions are well-documented and tested.
4. Avoid making breaking changes without discussing them first in an issue or pull request.
